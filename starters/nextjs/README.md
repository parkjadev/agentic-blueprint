# Next.js Starter

Full-stack Next.js starter extracted from AccessFit247 production patterns.

## Stack

- Next.js (App Router, strict TypeScript)
- Clerk + JWT dual auth
- Drizzle ORM + Neon (serverless Postgres)
- Inngest (background jobs)
- Upstash (rate limiting)
- Vercel (deployment)
