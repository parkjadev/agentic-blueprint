# Next.js Starter

Full-stack Next.js starter extracted from production patterns.

## Stack

- Next.js (App Router, strict TypeScript)
- Supabase (Auth, PostgreSQL, Storage)
- Drizzle ORM (type-safe queries)
- Inngest (background jobs, opt-in)
- Vercel (deployment)
