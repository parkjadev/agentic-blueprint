# Flutter Starter

Mobile companion starter with Riverpod state management, GoRouter navigation, and themed widgets.

## Stack

- Flutter (Material 3)
- Riverpod + Hooks
- GoRouter
- Dio (HTTP client for Next.js API)
- Supabase Auth (via supabase_flutter)
- Freezed + json_serializable
