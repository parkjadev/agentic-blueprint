# Agentic Blueprint

Opinionated full-lifecycle accelerator for building products with Claude as your primary collaborator. Based on production patterns from ARK360 ventures.

## Structure

- `docs/templates/` — Reusable document templates (PRD, specs, architecture)
- `docs/guides/` — Full lifecycle workflow guides
- `starters/nextjs/` — Full-stack Next.js starter
- `starters/flutter/` — Mobile companion starter
- `claude-config/` — Claude Code configuration templates
